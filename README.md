# Agentes-de-IA-para-monitorear-la-calidad-del-aire-con-Python-y-Copilot
Un agente de IA que monitorea la calidad del aire (PM10) en tiempo real. Con Python entrenamos un modelo que clasifica riesgo y detecta anomalías; luego Copilot en Power Automate genera resúmenes ejecutivos y envía alertas, sin conectores premium. Verás el flujo completo: datos “casi reales”, puntuación, etc.
